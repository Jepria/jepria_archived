      • rollback

        public void rollback()
        Откат (rollback) транзакции.
        В случае возникновения ошибки исключение не выбрасывается, а лишь записывается в лог. Откат сам по себе происходит в результате некоей ошибки. Если вызов отката привёл ещё к одному исключению, не остаётся уже ничего, кроме как закрыть соединение.

      • createConnection

        private static java.sql.Connection createConnection(java.lang.String dataSourceJndiName)
        Фабричный метод, создающий соединение JDBC. Поиск источника данных JDBC осуществляется по JNDI. Найденный источник заносится в хэш-таблицу и при повторном вызове берётся из неё, поэтому метод имеет модификатор synchronized.
