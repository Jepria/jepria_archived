        public void handle(java.lang.Throwable caught)
            throws ApplicationException
        Стандартная реализация метода, выполняемого после завершения транзакции.
        Выполняет следующие действия:
        • Если в ходе транзакции не возникло исключения, то транзакция фиксируется (CallContext.commit()).
        • Если было перехвачено исключение (caught != null), транзакция откатывается (CallContext.rollback()).
        • С помощью CallContext.end() освобождаются ресурсы.
        • Если в ходе транзакции возникло исключение, или же оно возникло во время commit либо rollback, выбрасывается последнее возникшее исключение.
        Specified by:
        handle in interface EndTransactionHandler
        Parameters:
        caught - перехваченное исключение
        Throws:
        ApplicationException - последнее выброшенное исключение
