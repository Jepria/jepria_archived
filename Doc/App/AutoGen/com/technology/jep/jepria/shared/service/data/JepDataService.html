prepareDownload

java.lang.Integer prepareDownload(java.lang.String fileName,
                                  java.lang.String mimeType,
                                  java.lang.String fieldName,
                                  java.lang.String recordKey,
                                  java.lang.String contentDisposition,
                                  java.lang.String extension,
                                  java.lang.String fileNamePrefix)
Записывает в сессию атрибуты для выгрузки файла и возвращает id загрузки.
Если значение fileName пусто, то имя выгружаемого файла формируется как fileNamePrefix + recordKey + "." + extension.

Parameters:
fileName - имя файла
mimeType - mime-тип
fieldName - имя поля в таблице
recordKey - ключ записи
contentDisposition - значение заголовка Content-disposition
extension - расширение
fileNamePrefix - префикс
Returns:
id загрузки
