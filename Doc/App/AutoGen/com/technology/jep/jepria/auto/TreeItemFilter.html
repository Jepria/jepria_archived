Interface TreeItemFilter


  • public interface TreeItemFilter
    Вспомогательный интерфейс (паттерн команда), используемый при обходе дерева. Методы инфтерфейса применяются ко всем узлам дерева во время обхода.
    Author:
    RomanovAS

    • Method Detail

      • putToResult

        boolean putToResult(TreeItemWebElement treeItem)
        Метод, проверяющий, нужно ли данный узел дерева помещать в результирующий список. Например, если нужно получить только отмеченные развернутые папки в дереве: 

         return treeItem.isExpanded() && treeItem.getCheckedState() == CheckedState.CHECKED;
        Parameters:
        treeItem - проверяемый узел дерева

      • traverseDescendants

        boolean traverseDescendants(TreeItemWebElement treeItem)
        Метод, проверяющий, нужно ли рекурсивно обходить детей данного узла, либо остановить обход вглубь на данном узле. Например, если не нужно обходить папки 3-го и больших уровней вложенности в дереве: 

         return treeItem.getLevel() < 3;
 

        
 Метод вызывается только для развернутых папок и, если имеется, корневого элемента.
        Parameters:
        treeNode - проверяемый узел
