selectComboBoxMenuItemWithCharByCharReloadingOptions

void selectComboBoxMenuItemWithCharByCharReloadingOptions(java.lang.String comboBoxFieldId,
                                                          java.lang.String menuItem,
                                                          int minInputLength)
Сложный выбор элемента Комбо-бокса по заданному id. Предполагается, что при первой загрузке опций не все опции появляются в списке (например, список слишком длинный), и искомая опция может стать доступной о мере ввода текста. Если искомой опции нет в выпадающем списке после введения всего текста, выбрасывается исключение WrongOptionException.

Parameters:
comboBoxFieldId - id ComboBox-поля
menuItem - Имя опции, которую необходимо выбрать
minInputLength - Минимальное количество символов, которые необходимо ввести для того, чтобы в данном Комбо-боксе началась загрузка опций (например, при поиске оператора, часто опции загружаются при вводе мимнимум 3 символов). Минимально допустимое значение = 1 (устанавливается автоматически при задании меньшего значения).

selectTreeItems

void selectTreeItems(java.lang.String treeFieldId,
                     java.lang.String[] itemPaths)
Выбор элементов JepTreeField по заданным путям, без предварительной очистки уже отмеченных узлов.

Parameters:
treeFieldId - id JepTree-поля
itemPaths - массив непустых путей опций, которые следует отметить в дереве. В случае если в дереве отсутствует хотя бы одна из требуемых опций либо промежуточный узел в пути, выбрасывается исключение WrongOptionException. В случае если входной параметр некорректен, выбрасывается IllegalArgumentException Синтаксис путей опций: {FolderName/}OptionName или {FolderName/}>FolderName; '/' в именах элементов нужно экранировать: '\/'. Предполагается, что имена элементов дерева не начинаются с '>' (иначе нужно внедрять экранирование этого символа). Предполагается, что нигде в дереве рядом не находится двух одноименных листьев или папок (но допускаются лист и папка с одним именем). Символ '>' может стоять перед именем последнего (подлежащего отмечанию) элемента в том случае, если это папка. Это различает ситуации, когда в дереве рядом находятся одноименные лист и папка.

getGridDataRowwise

java.util.List<java.util.List<java.lang.Object>> getGridDataRowwise(java.lang.String gridId)
Returns:
a list of rows, where each row is a list of cell-objects, or empty list if there is no grid (or it is empty). Each object in the resultant grid may be either of a type WebElement (if a cell contains something inside, such as input) or of a type String (if there is only a text). The value of a particular cell can be obtained using the row number and a column index, known from the result of #getGridHeaders().
If an object is a WebElement with a tagname 'tag', then its position in the grid as DOM is:
 <tbody id='gridId_BODY'>
   <tr>
     <td>
       <div>
         <tag/>
       </div>
     </td>
     ...
   </tr>
   ...
 </tbody>
