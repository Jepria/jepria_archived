Package com.technology.jep.jepria.client Description

Пакет содержит клиентскую (работающую в браузере пользователя) часть функционала.
Пакет поддерживает Free Layout элементов пользовательского интерфейса на клиентской форме.

Основная идея:
На пользовательских формах можно располагать любые Gxt(GWT)-компоненты (смотри Gxt Showcase). Работа с этими компонентами в плане "дизайнерской части" (работа НЕ связанная с данными) осуществляется через интерфейс view доступный в презентере.

Вся работа связанная с данными, как работа с данными для группы полей (детальная форма), так и работа со списками (списочная форма), ведется через интерфейсы, соответственно, fields и list . Эти интерфейсы также доступны в соответствующих презентерах.
Эти интерфейсы реализуются наследниками, соответственно, FieldManager и ListManager.

ВНИМАНИЕ!!! Важный момент:
Интерфейсы fields и list в презентерах иннициализируются *Manager'ами по умолчанию.
Но, *Manager-ов может быть произвольное количество. Их можно создавать столько, сколько (например) объединений полей или списков используется на формах. В этом случае *Manager-ы создаются во *View (обычным способом по new...) и "проключаются" в презентер посредством интерфейса view (например, делаем соответствующие геттеры).

Примеры реализации для простейшего случая форм - смотри в JavaDoc документации к пакетам com.technology.jep.jepria.client.ui.form.detail и com.technology.jep.jepria.client.ui.form.list .
