getFieldValueFromOption

protected <X> X getFieldValueFromOption(java.lang.String fieldId)
Получение значения опции JepOption из поля возвращающего опцию.
В реализации метод вызывает JepOption.getValue(Object option).
Пример использования в презентере прикладного модуля: 
   ...
   fields.<Integer>getFieldValueFromOption(CITY_ID); // Получение значения типа Integer или null.
   ...
   fields.<String>getFieldValueFromOption(COMPANY_CODE); // Получение значения типа String или null.
   ...

Parameters:
fieldId - поле
Returns:
значение опции поля
