isAcceptableWorkstate

protected boolean isAcceptableWorkstate(WorkstateEnum workstate)
Метод проверки соответствия рабочего состояния и текущего презентера.
Особенности:
В общем случае, смена состояния презентера должна происходить всегда. Однако в случаях списочной и детальной форм, фильтрация отдельных состояний позволяет существенно повысить производительность путем сокращения числа "лишних" срабатываний презентера.
Также для обработки специфичных прикладному модулю случаев, достаточно переопределить метод с необходимой бизнес-логикой.
Обрабатываем лишь специфичные для детальной формы состояния.

Overrides:
isAcceptableWorkstate in class JepPresenter<E extends PlainEventBus,F extends StandardClientFactory<E,S>>
Parameters:
workstate - рабочее состояние
Returns:
true - если рабочее состояние применимо для текущего презентера
