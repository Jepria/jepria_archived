implements com.google.gwt.user.client.ui.HasEnabled, com.google.gwt.user.client.ui.HasValue<java.util.Date>, com.google.gwt.editor.client.IsEditor<com.google.gwt.editor.client.LeafValueEditor<java.util.Date>>

Аналог GWT-класса DateBox, в котором виджет TextBox заменён на MaskedTextBox.
Имеет следующие особенности:

  • Виджет TextBox заменён на MaskedTextBox.
  • Для парсинга даты используется класс MaskedDateBox.XDefaultFormat, в котором дата парсится с помощью метода parseStrict(), что исключает значения наподобие 31.02.2013 55:66:77.
  • В методе setValue() вместо DateChangeEvent вызывается событие ValueChangeEvent.
  • Во всех методах вместо интерфейса DateBox.Format используется класс MaskedDateBox.XDefaultFormat.
  • Спецификатор доступа метода parseDate(boolean) изменён с private на protected.
  • Значение параметра fireNullValues по умолчанию true, а не false, что необходимо для корректной обработки обнуления даты с клавиатуры.
  • Стиль в конструкторе задаётся не с помощью setStyleName(), а с помощью addStyleName().
  • Конструктор принимает дополнительный параметр - маску..
