      • setCheckStyle

        public void setCheckStyle(TreeField.CheckCascade checkCascade)
        Устанавливает стиль поведения (каскадного выделения) для отмечаемых узлов.
        Возможные значения:
        • NONE - каскадного выделения никаких узлов не происходит
        • PARENTS - каскадно выделяются все родители
        • CHILDREN - каскадно выделюятся все дети
        Замечание: при установке значения CHILDREN отмечаются (очевидно) только отрисованные дочерние узлы.
        Parameters:
        checkCascade - стиль поведения (каскадного выделения) для отмечаемых узлов

      • setCheckNodes

        public void setCheckNodes(TreeField.CheckNodes checkNodes)
        Устанавливает какие узлы можно отмечать.
        Возможные значения:
        • BOTH - можно отмечать и узлы и конечные листья
        • PARENT - только родительские узлы (узлы, которые содержат дочерние элементы)
        • LEAF - только листья (узлы, которые НЕ содержат дочерних элементов)
        Parameters:
        checkNodes - какие узлы можно отмечать
