Class JepScopeStack

java.lang.Object
  com.technology.jep.jepria.client.history.scope.JepScopeStack

public class JepScopeStack
extends java.lang.Object

Состояние приложения.
Объект класса создается как singleton на уровне приложения и описывает текущее состояние приложения.
Класс является одним из основных элементов поддержки History в приложениях JepRia (смотри описание в com.technology.jep.jepria.client.history).
