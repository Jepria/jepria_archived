public abstract class JepBaseNumberField<E extends com.google.gwt.user.client.ui.ValueBox<? extends java.lang.Number>>
extends JepBaseTextField<E>

Поле для ввода произвольных чисел. Вводимыми символами данного поля могут быть только цифры (запрет ввода точек для целочисленных типов и прочих спецсимволов) Если необходимо запретить ввод отрицательных значений, то это делается методом setAllowNegative(false) Особенности:
Необходимость данной абстракции обусловлена тем, чтобы в прикладном коде избавиться от потребности генерализации поля JepNumberField

setViewValue

protected void setViewValue(java.lang.Object value)
Установка значения для карты Просмотра.

Если для поля задано значение формата вывода чисел, то значение устанавливается в соответствии с этим форматом. В противном случае выводится строковое значение, возвращаемое методом toString().

При перегрузке данного метода в наследниках необходимо обеспечить, чтобы данный метод был быстрым/НЕ ресурсо-затратным.

Основная идея Jep-полей: они должны быть легкими. Поэтому, карта Просмотра должна быть именно текстовым (или простым Html) представлением значения карты Редактирования.
В тех случаях, когда чисто текстовое представление нецелесообразно (списки, деревья и т.п.) - в поле используется ТОЛЬКО одна карта - карта Редактирования (т.е. карта Просмотра - вообще НЕ используется).
