  • java.lang.Object

  • public class JepScope
extends java.lang.Object
    Уровень иерархии модулей.
    Объект класса описывает состояние модулей приложения относящихся к одному уровню иерархии модулей.
    Класс является одним из элементов поддержки History в приложениях JepRia (смотри описание в com.technology.jep.jepria.client.history).

      • toHistoryToken

        public java.lang.String toHistoryToken(WorkstateEnum workstate)
        Метод формирует строковое представление (History Token) одного уровня иерархии модулей (одного scope'а) (объекта JepScope).

        Формат формируемой строки:
        • JepPlace.SCOPE_MODULES_PARAMETER_NAME - наименование параметра содержащего список модулей данного уровня
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • список идентификаторов модулей данного уровня перечисленных через запятую «,»
        • JepHistoryConstant.SCOPE_PARAMETER_SEPARATOR - разделитель между параметрами
        • JepPlace.WORKSTATE_PARAMETER_NAME - наименование параметра содержащего список состояний модулей данного уровня
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • список состояний модулей данного уровня перечисленных через запятую «,»
        • JepHistoryConstant.SCOPE_PARAMETER_SEPARATOR - разделитель между параметрами
        • JepPlace.ACTIVE_MODULE_PARAMETER_NAME - наименование параметра содержащего индекс активного модуля
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • индекс активного модуля
        • JepHistoryConstant.SCOPE_PARAMETER_SEPARATOR - разделитель между параметрами
        • JepPlace.SEARCH_REQUEST_PARAMETER_NAME - наименование параметра содержащего внешний ключ для данного уровня
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • Строка представляющая внешний ключ.
          Строка формируется методом JepHistoryToken.getMapAsToken(Map map)
        • JepHistoryConstant.SCOPE_PARAMETER_SEPARATOR - разделитель между параметрами
        • JepPlace.TEMPLATE_PROPERTIES_PARAMETER_NAME - наименование параметра содержащего поисковый шаблон для данного уровня
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • Строка представляющая поисковый шаблон.
          Строка формируется методом JepHistoryToken.getMapAsToken(Map map)
        • JepHistoryConstant.SCOPE_PARAMETER_SEPARATOR - разделитель между параметрами
        • JepPlace.PRIMARY_KEY_PARAMETER_NAME - наименование параметра содержащего последний полученный первичный ключ
        • JepHistoryConstant.SCOPE_PARAMETER_NAME_VALUE_SEPARATOR - разделитель между наименованием и значением параметра
        • Строка представляющая последний полученный первичный ключ.
          Строка формируется методом JepHistoryToken.getMapAsToken(Map map)
        Пример: sm=Site,Page,Host&ws=vd,null,vd&am=2&sr=siteId:33674&tp=siteId:33674&pk=hostId:522
        Parameters:
        workstate - новое рабочее состояние (режим) текущего активного модуля
        Returns:
        строковое представление (History Token) одного уровня иерархии модулей (одного scope'а) (объекта JepScope)
